# Robotxt

Robots.txt parser.

## Installation

```elixir
def deps do
  [
    {:robotxt, "~> 0.1.2"}
  ]
end
```

Documentation can be found at [https://hexdocs.pm/robotxt](https://hexdocs.pm/robotxt).
